# 🤖 AI Assistant

### Local AI Chatbot powered by Ollama (Private • Fast • Offline)

---

## 🚀 Overview

AI Assistant is a locally running AI chatbot built using Python and Ollama, designed to deliver fast, private, and intelligent conversations without relying on cloud APIs.

Unlike traditional chatbots, this assistant runs entirely on your machine using local Large Language Models (LLMs), ensuring **data privacy**, **low latency**, and **offline capability**.

---

## ✨ Key Highlights

* 🔒 **100% Local AI** – No internet required after setup
* ⚡ **Low Latency Responses** – Powered by Ollama runtime
* 🧠 **LLM Integration** – Supports Llama2 / Mistral / Gemma
* 💬 **Real-time Chat Interface (CLI)**
* 🔄 **Continuous Conversation Loop**
* 🧩 **Modular & Extensible Architecture**

---

## 🧠 How It Works

1. User inputs a query via terminal
2. Python script sends request to Ollama
3. Ollama processes it using a local LLM
4. Response is returned and displayed instantly
